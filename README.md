## Dependencies

- Python >= 3.7.5 
- Numpy  >= 1.18.4

## How to use

```sh
$ python HWxx_0416047.py xxx.ooo
```
- `HWxx_0416047.py` is python script in the HWxx directory.
- `xxx.ooo` is SP3-c or RINEX file.
- Make sure that python script and SP3-c/RINEX file are places in the same directory.
